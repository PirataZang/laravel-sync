# 💰 Financial Manager

Um sistema moderno de **gestão financeira pessoal**, desenvolvido para ajudar usuários a **controlar receitas, despesas e acompanhar sua saúde financeira de forma simples e visual**.

A ideia do projeto é trazer para um **sistema web e mobile** o conceito das famosas **planilhas financeiras**, mas com uma experiência muito mais intuitiva, automatizada e visual.

O sistema permite registrar movimentações financeiras, visualizar gráficos, acompanhar relatórios e entender rapidamente se o mês terminou **no verde ou no vermelho**.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando um stack moderno focado em performance, escalabilidade e organização do código.

- **Backend**
  - Laravel
  - PostgreSQL

- **Frontend**
  - Vue.js

- **Infraestrutura**
  - Docker
  - Docker Compose

---

## 🎯 Objetivo do Projeto

O objetivo deste sistema é permitir que qualquer pessoa consiga **organizar sua vida financeira de maneira simples e clara**, oferecendo ferramentas que facilitem o acompanhamento do dinheiro no dia a dia.

Diferente de planilhas tradicionais, o sistema:

- Automatiza cálculos
- Apresenta **gráficos intuitivos**
- Gera **relatórios personalizados**
- Permite **filtrar informações rapidamente**
- Funciona **tanto em desktop quanto em dispositivos móveis**

---

## 📊 Funcionalidades

### 💵 Cadastro de Movimentações
- Registro de **receitas**
- Registro de **despesas**
- Categorias personalizadas
- Descrição e observações

---

### 📈 Dashboard Financeiro
Uma tela central com as principais informações financeiras do usuário:

- Resumo do mês
- Total de receitas
- Total de despesas
- Saldo atual
- Indicador visual se o mês está **positivo ou negativo**
- Gráficos de distribuição de gastos

---

### 📅 Filtros Inteligentes
Permite filtrar informações para análises mais detalhadas:

- Por período
- Por categoria
- Por tipo de movimentação
- Por mês ou ano

---

### 📑 Relatórios Personalizados
O sistema permite gerar relatórios financeiros personalizados com base nos filtros selecionados.

Exemplos:

- Relatório mensal
- Relatório por categoria
- Relatório de despesas
- Relatório completo

Os relatórios podem ser **baixados para análise externa**.

---

### 📊 Visualização com Gráficos
Para facilitar o entendimento dos dados financeiros, o sistema apresenta gráficos como:

- Distribuição de gastos
- Evolução financeira
- Comparação entre receitas e despesas

---

A interface é responsiva, garantindo uma boa experiência em qualquer dispositivo.
